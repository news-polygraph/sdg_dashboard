# start local

to run Backend:

1. Install python 3.9
2. Create venv
3. install requirements.txt
4. run: python -m spacy download en_core_web_sm

-> start backend: python backend/base.py

to run Frontend:

1. Install node v22.9.0
2. run npm install

-> start frontend: npm start

# start within docker

docker-compose up
